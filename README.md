# Network Routing Optimisation using Genetic Algorithm

## Overview
This project implements a genetic algorithm (GA) to optimise routing in a network. The goal is to find the most efficient path between nodes based on distance (cost).

## Features
- Simulates a network of nodes
- Uses Genetic Algorithm for optimisation
- Evaluates route performance using a cost function

## Tools
- Python

## Outcome
The model identifies efficient routes in a constrained network, demonstrating how optimisation algorithms can be applied to routing problems.
